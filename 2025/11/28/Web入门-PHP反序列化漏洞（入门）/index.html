Web入门-PHP反序列化漏洞（入门）

序列化是指将对象或者其他变量转换为一种可保存或传输的字符串形式的过程；

而反序列化相反，会将这个字符串重新转换回原来的变量形式或者对象以供使用。

解析

O:数据类型，这里表示object对象，用来表示包含多个属性的

​ 相应的，s代表单纯的字符串str，i代表整形数字int，也就是后面的s和i含义

“user”:2: ：这个对象的名字，后面跟着的2代表这个对象含有两个属性

在{}内的数据代表这个对象的属性

比较需要注意的一点是，不同的数据类型必须定义相对应的数据

PHP中的魔术方法调用

php的反序列化最关键的部分，大部分题目都是需要调动魔术方法来进行解题

这类题型可能会出比较难的题目

主要讲解几个魔术方法

call

它在尝试调用一个对象中不可访问的方法时被调用。

toString

被当作字符串处理时自动调用该方法，并返回该对象的一个字符串表示。

invoke

允许一个对象像函数那样被调用。当你尝试以调用函数的方式调用一个对象时（例如，$obj()），__invoke() 方法会被自动调用。

wakeup/sleep

分别在对象或者变量被序列化或者反序列化的时候调用
